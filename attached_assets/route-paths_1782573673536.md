---
name: Route paths and nav links
description: The app uses short route slugs /find and /ai, not /find-benefits or /ai-assistant.
---

Actual routes (from App.tsx):
  / → Home
  /find → FindBenefits (find-benefits.tsx)
  /ai → AiAssistant (ai-assistant.tsx)
  /schemes → Schemes
  /schemes/:id → SchemeDetail
  /saved → SavedSchemes
  /profile → Profile
  /settings → Settings
  /about → About
  /contact → Contact

**Why:** Short slugs were chosen when the routes were first set up. Subagents and screenshots often navigate to /find-benefits or /ai-assistant which returns 404.

**How to apply:** Use /find and /ai everywhere — in Link hrefs, screenshots, tests, and any internal redirects.
