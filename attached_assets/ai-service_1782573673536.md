File: artifacts/api-server/src/services/ai.ts

Pattern:
  const openai = process.env.OPENAI_API_KEY ? new OpenAI({...}) : null;
  generateAIResponse() checks if openai is null → falls back to generateMockResponse()

Model used: gpt-4o-mini (chat completions, max_tokens: 1024)
System prompt: SYSTEM_PROMPT constant in ai.ts — describes the Benefits Advisor role

Conversation history: last 20 messages fetched and passed to OpenAI for multi-turn context

**Why:** App must work without any API key so users can try it before setting up OpenAI.

**How to apply:** Set OPENAI_API_KEY secret to enable real AI. No code changes needed — the service auto-detects it at startup. To change the model, edit the model string in generateAIResponse().
